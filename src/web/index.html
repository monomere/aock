aock

tiny risc-v operating system for christmas fun!

features

  • small
  • risc-v
  • not unix
  • message based

language

userspace programs (actors) use a builtin smalltalk-style language, since the way processes work is not really compatible with most systems languages being imperative. it's also fun to make a compiler for risc-v and learn stuff c:

the language is not whitespace sensitive, uses kebab-case. compiles to risc-v. 

# small example
on Startup do
  uart/print to: 0 text: "hi!"
  os/exit :successfully
end
